AFABL Programmer Study

I am integrating reinforcement learning into a Scala-embedded domain-specific language (DSL). This language, which I call AFABL (A Friendly Adaptive Behavior Language), makes it possible to create intelligent software agents using elements of the reinforcement learning model. I claim that writing particular kinds of intelligent software agents is easier in AFABL than in traditional programming languages and that AFABL agents are easier to adapt to new environments. Scientifically supporting these claims is the final piece of my doctoral dissertation in computer science at Georgia Tech.
